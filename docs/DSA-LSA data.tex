The model is calibrated to Greece using annual data for 1995--2024 from Eurostat, OECD, and national sources. The calibration period covers the pre-crisis expansion, the sovereign debt crisis and adjustment programmes, and the post-pandemic recovery. See Appendix~\ref{s:calibration} for data sources, moment definitions, and the estimation algorithm.

\subsection{Strategy}

Calibration proceeds in two stages. In the first stage, parameters that can be measured directly from data or taken from the literature are set externally without solving the model. In the second stage, parameters that govern savings behaviour and the income process are calibrated internally by minimising the distance between simulated and empirical moments, as described in Appendix~\ref{s:calibration}.
